# WebCraft Pro (Next.js + Tailwind)

## วิธีรัน

```bash
npm install
npm run dev
```

## วิธี build

```bash
npm run build
```

## หมายเหตุ

- อนุญาตโดเมนรูปภาพ `static.photos` ผ่าน `next.config.mjs`
- เตรียมพร้อมเชื่อม Supabase ในอนาคต โดยสามารถเพิ่มค่า env เช่น `NEXT_PUBLIC_SUPABASE_URL` และ `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## GitHub Setup

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <repo_url>
git push -u origin main
```
